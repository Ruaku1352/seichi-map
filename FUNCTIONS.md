# アプリ機能一覧（FUNCTIONS.md）

ユーザーが実際に触れる機能をカテゴリ別にまとめたドキュメント。
実装の詳細（コンポーネント名・state 名など）は STRUCTURE.md を参照。

---

## 1. 地図

| 機能 | 詳細 |
|---|---|
| カスタムマップスタイル | 黄色い道路・青い水域・緑の公園など、アニメらしい柔らかいカラーパレット |
| 移動制限 | 日本国内（北緯 23〜46.5°、東経 121〜155°）から地図が出ないよう制限 |
| 初期表示 | 起動時は東京付近（zoom 6）を表示 |
| ジェスチャー操作 | スワイプ・ピンチ・ダブルタップで自由にパン・ズーム |

---

## 2. 聖地ピン

| 機能 | 詳細 |
|---|---|
| 35件の聖地表示 | 紫の星ピンで地図上に表示 |
| クラスタリング | ズームアウト時に近い聖地をまとめて数字バッジで表示。ズームインで分散 |
| 選択中ピン | 選択中の聖地は紫でパルスアニメーション（500ms ごとに拡縮） |
| 検索ヒット時の強調 | 検索でヒットした聖地のみ赤の縦長ピンで表示。その他はグレーに減光 |

---

## 3. 観光スポットピン

| 機能 | 詳細 |
|---|---|
| 25件の観光スポット表示 | 関西 10 件・関東 15 件をオレンジの丸ピンで表示（通天閣・浅草寺など） |
| タップでポップアップ | スポット名・英語名を簡易ポップアップで表示。地図タップまたは ✕ で閉じる |

---

## 4. 聖地カード

| 機能 | 詳細 |
|---|---|
| 折りたたみ表示 | ヘッダー部（アニメ名・スポット名・エリア・距離）は常時表示。タップで本文を展開 |
| 現在地からの距離 | GPS または仮想マーカーからの直線距離を表示（例: About 1.2km） |
| AI 生成紹介文 | Claude Haiku が生成した英語紹介文を展開エリアに表示 |
| ローディング表示 | 紹介文生成中は小さなスピナーと "Generating introduction…" を表示 |
| AI / 静的バッジ | 生成成功時は「✨ AI generated」、失敗時は「📄 description」を表示 |
| 閉じるボタン | カード右上の ✕ ボタンで閉じる |

---

## 5. AI 紹介文生成

| 機能 | 詳細 |
|---|---|
| リアルタイム生成 | カードを開くたびにバックエンド（FastAPI + Claude Haiku）へリクエストを送信 |
| パーソナライズ | ユーザーの好み（nickname / familiarity / mood / travelStyle）に合わせてプロンプトを調整（詳細は「7. パーソナライズ」参照） |
| フォールバック | 生成失敗時は `generic_intro_en`（カード別汎用文）または `intro_short_en` を表示。どちらも未設定なら汎用英語テキストを表示 |
| キャッシュ | 生成済みの紹介文はクライアント変数・サーバーメモリの両方にキャッシュされ、同じスポットの2回目以降は API を呼ばない（prefs なし時のみ） |

---

## 6. LIVE モード（デフォルト）

| 機能 | 詳細 |
|---|---|
| GPS 追跡 | `watchPosition` でリアルタイムに現在地を取得し、青い丸マーカーで地図に表示 |
| 位置取得中のローディング | GPS 取得が完了するまで（最大 8 秒）全画面の白いローディング画面を表示 |
| カードの手動表示 | 聖地ピンをタップするとカードを表示 |
| カメラ追従 | カード表示時はそのスポットへカメラがズーム。カードを閉じてもカメラはその場に留まる（自動で元の位置に戻らない） |
| GPS ボタン | 画面右下に表示。GPS 取得状況を色で表示し、タップで現在地へカメラを移動 |
| DEMO への切り替え | 左下コントロールバーの「DEMO」ボタンで DEMO モードに切り替え |

---

## 7. DEMO モード

| 機能 | 詳細 |
|---|---|
| LIVE への切り替え | 左下コントロールバーの「LIVE」ボタンで LIVE モードに切り替え |
| 開始地点設定 | 「📍 Set Start」ボタン押下後、地図をタップして仮想マーカーの出発点を設定 |
| 変更ヒント | デモモード起動直後、Set Start を促す吹き出しが表示される |
| 自動ルート生成 | 開始地点から半径 40km 以内の聖地を、近い順に自動でルート設定 |
| 再生 / 一時停止 | ▶️ / ⏸ ボタンで仮想マーカーの移動を制御（開始地点未設定時はボタンが無効） |
| 仮想マーカー移動 | 紫の丸マーカーが 600ms ごとに約 110m ずつ自動移動 |
| 近接判定（自動カード表示） | 仮想マーカーが聖地から 100m 以内に入るとカードを自動表示 |
| 自動カードクローズ | 仮想マーカーが聖地から 100m 以上離れるとカードを自動クローズ |
| スポット到達後の停止 | 各スポット到達後は 5 秒停止してから次のスポットへ移動 |
| カメラ追従 | 仮想マーカーを常にカメラが追跡 |

## 8. 検索機能

| 機能 | 詳細 |
|---|---|
| 検索バー | ヘッダー右側（"Search here"）にアニメ名を入力 |
| 候補表示 | 入力に部分一致するアニメ名を最大 6 件ドロップダウン表示 |
| ピン強調 | アニメを選択するとそのアニメの聖地のみ赤ピンで表示。他の聖地はグレーに減光 |
| バウンスアニメーション | 検索直後、ヒットしたピンが 2.5 秒間バウンスして場所を知らせる |
| リセット | 検索バー右の ✕ ボタンで検索クリア → 元の全ピン表示に戻る |
| キーボード操作 | Esc キーで候補を閉じる |

---

## 9. 初回アンケート

| 機能 | 詳細 |
|---|---|
| 表示条件 | 初回起動時（localStorage に好みデータがない場合）のみ全画面表示 |
| ステップ 1 | ニックネーム入力（任意。後でカードの呼びかけに使用） |
| ステップ 2 | アニメへの馴染み度（Newcomer / Casual fan / Big fan） |
| ステップ 3 | 今の気分（Emotional / Exciting / Heartwarming / Romance） |
| ステップ 4 | 旅のスタイル（Taking photos / Relaxed walking / Visiting many spots） |
| 保存 | 入力完了後に localStorage へ保存。次回以降はアンケートを省略 |

---

## 10. 設定画面

| 機能 | 詳細 |
|---|---|
| 開き方 | 左下コントロールバー左端の ⚙️ ボタン |
| 変更可能な項目 | nickname / familiarity / mood / travelStyle（アンケートと同じ 4 項目） |
| 保存 | 「Save」ボタンで localStorage を更新。カードキャッシュを即時クリアし、次のカードから新しい設定が反映される |
| リセット | 「Reset」ボタンで全設定を削除し、初回アンケートに戻る（確認ステップあり） |
| キャンセル | ✕ ボタンで変更なく閉じる |

## 11. パーソナライズ紹介文

ユーザーが設定した好みに応じて、Claude Haiku へのプロンプトが変わる。

| 設定 | 値 | 紹介文への影響 |
|---|---|---|
| `nickname` | 任意の文字列 | 紹介文の冒頭で名前を呼びかける（例: "Hey Taro,"） |
| `familiarity` | Newcomer | アニメ未知の読者向けに、前提知識なしで説明 |
| `familiarity` | Casual fan | 親しみやすく自然にシリーズを紹介 |
| `familiarity` | Big fan | 特定シーンやキャラクターに踏み込んだ深い解説 |
| `mood` | Emotional | 感動的・心を揺さぶる側面を強調 |
| `mood` | Exciting | 躍動感・興奮を強調 |
| `mood` | Heartwarming | 温かく穏やかな雰囲気を強調 |
| `mood` | Romance | ロマンチックな景観・シーンを強調 |
| `travelStyle` | Taking photos | 写真映えするポイントを提案 |
| `travelStyle` | Relaxed walking | ゆったり楽しむ方法を提案 |
| `travelStyle` | Visiting many spots | 立ち寄る価値のあるポイントを簡潔に紹介 |

> ⚠️ 好みが設定されている場合、クライアント・サーバー両方のキャッシュをバイパスして毎回生成する。

## 12. PWA 対応

| 機能 | 詳細 |
|---|---|
| ホーム画面追加 | iOS / Android のブラウザから「ホーム画面に追加」でアプリとして起動可能 |
